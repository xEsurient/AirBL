🔗 Gluetun Integration

Automatically generate a servers.json file for Gluetun, containing custom settings per profile such as endpoint routes, speed thresholds, and country filters.

Force Update
After writing servers.json, cycle the Gluetun VPN via its control server API to apply changes immediately

Control when the VPN connection is cycled after updating servers.json
Control Server Host
IP address of the Gluetun control server
Control Server Port
Port of the Gluetun control server (default: 8000)

Alternatively, you can export safe servers directly via API:

Export All Clean Servers:
curl -s http://localhost:5665/api/export/gluetun
Export Clean Servers by Country (e.g., US):
curl -s "http://localhost:5665/api/export/gluetun?country=US"
