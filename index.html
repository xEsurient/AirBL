DNS-Guided Endpoints

Uses direct DNS polling and rapid ICMP sampling to map accurate Entry and Exit IP pairs instantly without generating aggressive legacy subnet noise.

DroneBL Verification

Query every responsive server directly against the DroneBL blocklist using DNS to confirm its cleanliness before risking your traffic.

Encapsulated Speedtests

Seamlessly benchmark connection limits through isolated policy-routed containers without compromising the host network or exposing raw IP data.

Gluetun REST Integration

Automatically rebuild the optimal servers.json layout for Gluetun and issue live rolling restarts via REST directly to your download stack.

Dynamic Dashboard

Supervise scan history and system health metrics from a sleek, self-hosted web interface tracking connection anomalies and historical ban rates.

Built for Docker

Deploys locally effortlessly. Mount your AirVPN configurations, set your environment limits, and bring up the full stack natively via Compose.

RFC1918 Policy Routing

Intelligent Docker bridge bypassing routes UI traffic cleanly over the host while aggressive benchmark speedtests are seamlessly encapsulated via advanced local iptables.

Multi-Day Discovery

Automatically cycle sequential protocol combinations (1637, 47107) and map Entry endpoints over 7 days to scientifically guarantee the fastest localized VPN route structure.

Advanced Telemetry

Isolate overarching topological drops natively via long-term Metric dashboards tracking global DroneBL strikes, granular Entry Ping delays, and total historical Ban aggregations.
